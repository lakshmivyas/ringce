Overview & History

Slammer is a down to earth utility that displays a transparent grid that can be placed on top of applications during design.

The initial requirement came to me when I had alignment issues on my website and none of the CSS Tools (Xylescope, CSSEdit) came with a built-in grid overlay. Firefox, however, had an add-in called GridFox which became the inspiration for this tool.

Designing an iPhone App?
Run Slammer, drag the grid on top of any window. Push those pixels until they line up with the grid.

Completely customizable

  • Change the background color and the grid color according to the application background so that the grid is prominent yet non-intrusive.
  • Change the grid dimensions to suit your applications.
  • Add multiple grids to align different aspects of your applications.

Position it with ease

Slammer has excellent keyboard support for precise positioning and alignment.

  • Position the grid using the arrow keys.
  • Resize the grid using shift + arrow keys.
